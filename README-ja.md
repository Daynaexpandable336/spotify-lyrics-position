# Lyrics Position

[English](README.md)

Spotify ([Spicetify](https://spicetify.app/)) の歌詞を、画面中央ではなく好きな位置に表示するための拡張機能。

## 機能

- 表示位置を自由に選択(設定は自動保存):
  - **右サイドバー** — 再生中ビュー(ジャケット・曲名・アーティスト名が出る右パネル)の最上部に歌詞カードを挿入
  - **左サイドバー** — ライブラリ(左パネル)の最上部に歌詞カードを挿入
  - サイドバーカードの高さは既定でセクションの約36%。設定で15〜80%に変更可能
  - **アプリ内ポップアップ** — ヘッダーをドラッグして移動、右下をドラッグしてリサイズ。位置・サイズは保存
  - **別ウィンドウ** — Spotify とは独立したウィンドウに表示
- **公式の歌詞ボタン(マイクアイコン)で、中央の歌詞画面の代わりにこの表示を開閉**(オフにすると従来の動作に戻る)
- 公式風の見た目: アルバムカラーの背景+未再生行は暗色、再生済み/現在行は白
- 同期歌詞のハイライト+自動スクロール(自動スクロールはオフにも可能)、行クリックでシーク
- 歌詞の取得元: Spotify 公式の歌詞 → 取得できない場合は [LRCLIB](https://lrclib.net/) に自動フォールバック
- 英語 / 日本語 UI — 既定では Spotify の言語設定に従う。設定で手動切替可能
- デフォルト設定に戻すボタン

## 前提条件

これは **Spicetify の拡張機能**です。先に [Spicetify](https://spicetify.app/) のインストールが必要です。
公式インストールガイド: https://spicetify.app/docs/getting-started

> [!WARNING]
> この拡張機能は **Windows** で開発・動作確認しています。macOS/Linux でも動作するはずですが、不安定な可能性があります。

## インストール

1. [Releases](../../releases) から最新の `lyrics-position.js` をダウンロード
2. Spicetify の `Extensions` フォルダに配置:

   | OS          | パス                             |
   | ----------- | -------------------------------- |
   | Windows     | `%APPDATA%\spicetify\Extensions` |
   | macOS/Linux | `~/.config/spicetify/Extensions` |

3. 以下を実行:

   ```
   spicetify config extensions lyrics-position.js
   spicetify apply
   ```

## 使い方

- 歌詞表示の開閉:
  - **公式の歌詞ボタン(マイクアイコン)** —「公式の歌詞ボタンで開閉する」がオン(既定)のとき
  - **この拡張機能が再生バーに追加するボタン** — 公式ボタンの乗っ取りをオフにしたときに表示される(オンの間はマイクアイコンが2つ並ばないよう非表示)
- 設定画面の開き方(表示をオフにしていても開ける):
  - 歌詞ボタン(マイクアイコン)を**右クリック**
  - 再生バーの追加ボタンを**右クリック**
  - 右上の**プロフィールメニュー →「Lyrics Position 設定」**
- 設定では表示位置・カード高さ・文字サイズ・自動スクロール・言語・歌詞の取得元を変更可能(即時反映・自動保存)
- 「公式の歌詞ボタンで開閉する」をオフにすると、マイクアイコンは従来どおり中央の公式歌詞画面を開く
- 「デフォルト設定に戻す」で全設定を即座に初期値に戻せる

## メモ

- 設定は Spotify の localStorage に保存されるため、アプリの再起動や `spicetify apply` 後も維持される(消えるのは Spotify のアプリデータ削除・再インストール時のみ)
- 右サイドバーモードで再生中ビューが閉じている場合は自動で開こうとする
- 別ウィンドウを閉じると表示モードは「表示しない」に戻る
- 別ウィンドウを常に最前面にしたい場合は、PowerToys の「常に手前に表示」(Win+Ctrl+T)が使える
- 歌詞を取得できない場合はカード内に理由(403 / 見つからず 等)を表示

## アンインストール

```
spicetify config extensions lyrics-position.js-
spicetify apply
```

末尾の `-` は誤記ではなく、Spicetify で「登録を削除する」ための公式構文。
